Computer Science

Computer science is the study of computers and other computational systems. It is a broad field that includes everything from how well software is developed and put together to how the algorithms that go into making software work with hardware. To address complex issues, computer scientists employ logical and analytical thinking, creativity, design, and programming skills.

Careers / Fields

Some careers that you can go into with a computer science degree include:

Data analysis
Network engineering
Software development
Software testing
